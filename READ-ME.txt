# Template para Academia EliteFit (Baseado em Fitness da HTML Codex)

Este é um template HTML/CSS/JS para sites de academias e centros de fitness, totalmente traduzido para o português e adaptado para o mercado brasileiro, com foco em SEO (Otimização para Mecanismos de Busca) para a cidade de São Paulo. O projeto foi desenvolvido para a academia fictícia "Academia EliteFit", utilizando como base o template "Fitness" da HTML Codex.

## 🚀 Como utilizar

Este é um projeto estático (HTML, CSS, JS) e não requer um processo de build com `npm`. Para visualizá-lo e editá-lo, siga os passos abaixo:

1.  **Clone o repositório ou baixe os arquivos:**
    ```bash
    git clone https://github.com/suaempresa/nome-do-projeto.git
    ```

2.  **Execute o projeto:**
    Não há dependências para instalar. Você pode abrir o projeto de duas maneiras:

    *   **Opção A (Simples):**
        Abra o arquivo `index.html` diretamente em seu navegador.

    *   **Opção B (Recomendado para Desenvolvimento):**
        Use um servidor web local para evitar problemas com requisições de arquivos (CORS). Se você usa o Visual Studio Code, a extensão **Live Server** é uma excelente opção.
        - Instale a extensão "Live Server".
        - Clique com o botão direito no arquivo `index.html` e selecione "Open with Live Server".

## ⚙️ Tecnologias

As tecnologias utilizadas neste template são:

-   **Estrutura:** HTML5, CSS3, JavaScript (ES6)
-   **Framework CSS:** Bootstrap 5
-   **Bibliotecas JavaScript:** jQuery, Owl Carousel, Wow.js (para animações de scroll), Easing.js, Waypoints.js
-   **SEO:** Otimização on-page, meta tags descritivas, semântica HTML e localização de conteúdo para São Paulo.

## 📝 Documentação

O código é comentado nas seções principais para facilitar a compreensão. A estrutura de arquivos segue um padrão simples:

-   `/css`: Contém os arquivos de estilo (CSS).
-   `/js`: Contém os arquivos de script (JavaScript).
-   `/img`: Contém as imagens utilizadas no projeto.
-   `/lib`: Contém as bibliotecas de terceiros (Owl Carousel, Wow.js, etc.).
-   `*.html`: Os arquivos das páginas do site estão na raiz do projeto (`index.html`, `about.html`, etc.).
